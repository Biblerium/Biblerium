Выражения с переменными

(Буквенные выражения).

1)  Выражение с переменными (буквенное выражение) состоит из чисел, переменных, знаков действий соединенных со смыслом.

2)  Значение выражения с переменными – число, которое получается в результате выполнения всех действий с выбранными значениями переменных.

 

Свойства действий над числами.

1) Переместительное свойство: для любых чисел a и b верны равенства:

a+b = b+a

ab = ba

2) Сочетательное свойство: для любых чисел a, b и c верны равенства:

(a+b)+c = a+(b+c)

(ab)c = a(bc)

3) Распределительное свойство: для любых чисел a, b и c верно равенство:

a(b+c) = ab+ac

4) В любой сумме можно как можно угодно переставлять слагаемые и произвольным образом объединять их в группы.

5) В любом произведении можно как угодно переставлять множители и произвольным образом объединять их в группы.

 

Тождества.

1)  Два выражения, значения которых равны при любых значениях которых равны при любых значениях переменных, называются тождественно равными.

2)  Равенство, верное при любых значениях переменных, называется тождеством.

3)  Замену одного выражения другим тождественно равным ему выражением, называют тождественным преобразованием или просто преобразованием выражения.

4)  Подобными слагаемыми называют слагаемые алгебраической суммы, имеющие одинаковую буквенную часть.

5)  Чтобы привести подобные слагаемые, надо сложить их коэффициенты и результат умножить на общую буквенную часть.

6)  Если перед скобками стоит знак плюс, то скобки можно опустить, сохранив знак каждого слагаемого, заключенного в скобки;                                                                                                    Если перед скобками стоит знак « - », то скобки можно опустить, изменив знак каждого слагаемого, заключенного в скобки, на противоположный.

 

 

 

 

 

 

Уравнения и его корни.

1)  Корнем уравнения с одной переменной называется такое значение переменной, при котором уравнение обращается в верное числовое равенство.

2)  Решить уравнение – значит найти все его корни или доказать, что корней нет.

3)  Если в уравнении перенести слагаемое из одной части в другую, изменив его знак, то получится уравнение, равносильное данному.

4)  Если обе части уравнения умножить или разделить на оно и то же отличное от нуля число, то получится уравнение, равносильное данному.

 

Линейное уравнение с одной переменной.

1)  Уравнение вида ax = b, где x – переменная, a и b – некоторые числа, называется линейным уравнением с одной переменной.

2)  Линейное уравнение ax – b при a ≠ 0 имеет один корень, при a = 0 и b = 0 имеет бесконечно много корней (любое число является корнем)

 

Модули.

Модуль – это расстояние от начала координат до точки, соответствующей числу на координатной прямой. Обозначается как « |x| », в котором x – число или выражение.

Функция.

Функция – зависимость одной переменной от другой переменной при которой каждому значению первой переменной соответствует единственное значение второй переменной.

 

Линейные функции.

1)  Графиком функции называется множество всех точек координат плоскости, абсциссы которых равны значениям аргумента, а ординаты – соответствующие значениям функции.

2)  Линейная функция – функция вида y = kꞏx+b, где                     x = независимая                                                                                         y = зависимая переменная                                                                                   k, b = числа

3)  Графиком линейной функции является прямая.

 

Прямая пропорциональность.

1)  Прямой пропорциональностью называется функция, которую можно задать формулой вида y = kꞏx, где x – независимая переменная, k – не равное нулю число.

2)  График прямой пропорциональности прямая, проходящая через начало координат. (точка (0;0)).

Одночлены.

1)  Произведение состоящее из числовых множителей–переменных, называется одночленом (т.е. только умножение)

2)  Произведение всех числовых множителей одночлена называется коэффициентом одночлена.

3)  Стандартным видом нулевого одночлена называется его запись, при которой:

a)   Коэффициент стоит на 1 месте;

b)  Каждая переменная участвует в записи одночлена лишь 1 раз в виде соответствующей степени;

c)   Буквы в записи одночлена (если они есть) следуют в алфавитном порядке.

4)  Степенью ненулевого одночлена называется сумма показателей степеней входящих в одночлен переменных.

5)  Одночлены, имеющие в стандартном виде одинаковую буквенную часть, называются подобными.

 

Функции.

1)  Областью определения функции является множество всех действующих чисел (x - любое) (x Q)

2)  Областью определения функции является множество всех действующих чисел.

3)  Если x = 0, то y = 0

4)  Если x > 0, то y > 0

5)  Если x < 0, то y < 0

6)  Противоположным значение x соответствуют противоположным значение y.

 

Многочлен.

1)  Многочлен – сумма одночленов.

2)  Если каждый член многочлена является одночленом стандартного вида и не содержит подобных членов, то такой одночлен называется многочленом стандартного вида.

3)  Степенью стандартного вида называется наибольшую из степеней входящих в него одночленов.

 

Формула сокращенного умножения (ФСУ).

1) (a+b)²= (a+b)ꞏ(a+b) = a² + 2ab + b²=формула квадрата суммы

    Читается как: Квадрат суммы двух выражений равен квадрату первого выражения плюс удвоенному произведению этих выражений плюс квадрат второго выражения.

2) (a-b)² = a² + 2ab + b² = формула квадрата разности

    Читается как: Квадрат разности двух выражений равен квадрату первого выражения минус удвоенное произведению этих выражений плюс квадрат второго выражения.

3) (a+b)³ = a³ + 3a²b + 3ab² + b³ = формула куба суммы

    Читается как: Куб суммы двух выражений равен, кубу первого выражения плюс утроенное произведение квадрата первого выражения на второе плюс утроенное произведение первого выражения на квадрат второго плюс куб второго выражения.

4) (a-b)³ = a³ - 3a²b + 3ab² - b³ = формула куба суммы

    Читается как: Куб разности двух выражений равен, кубу первого выражения минус утроенное произведение квадрата первого выражения на второе плюс утроенное произведение первого выражения на квадрат второго и минус куб второго выражения.

 

Торжества для ФСУ.

1) (a+b)² = (b+a)²

2) (a-b)² = (b-a)²

3) (-a-b)² = (a+b)²

Разность квадратов.

a²-b² = (a-b)ꞏ(a+b)

Читается как: Разность квадратов двух выражений равна произведению разности этих выражений на их сумму.

 

Сумма и разность кубов.

1) a³+b³ = (a+b)ꞏ(a²-ab+b²)

    Читается как: Сумма кубов двух выражений равна произведению суммы этих выражений и неполного квадрата их разности.

2) a³-b³ = (a-b)ꞏ(a²+ab+b²)

    Читается как: Разность кубов двух выражений равна произведению разности этих выражений и неполного квадрата их суммы.

Линейное уравнение с двумя переменными.

1) Линейным уравнением с 2 переменными называется уравнение вида:

        aꞏx+bꞏy = c

        x,y – переменные

2)Решением уравнения с 2 переменными называется пара значений переменных, обращающая это уравнения в верное числовое равенство.

 

График линейного уравнения с 2 переменными.
